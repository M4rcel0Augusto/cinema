# Sistema de Gestão de Cinema

## Status do Projeto:
Em desenvolvimento (alpha)

----

## Tecnologias Utilizadas:

* Java
* JPA
* Hibernate
* MySQL

## Time de Desenvolvedores:
Apenas um.

## Objetivo do Software:
O sistema tem como objetivo facilitar a gestão de operações básicas de um cinema.

## Funcionalidades:
### Gestão de Filmes e Sessões
- Cadastro de filmes.
- Cadastro de Sessões.

### Gestão de Produtos e Combos.
- Cadastro de produtos.
- Cadastro de combos.
- Controle de estoque de produtos.

### Vendas
- Registro de vendas de produtos e combos.
- Cálculo automático do valor total da venda.
- Atualização automática do estoque após a venda.

  ### Relatórios
  - Relatórios de receita, custo e lucro de cada mês.

  ### Alerta de Estoque Mínimo
  - Possibilidade de definir a quantidade mínima de um produto em estoque.
  - Quando esse valor for atingido, será disparado um alerta.
 
  ## Banco de Dados
  O arquivo text.sql está disponível no repositório.
